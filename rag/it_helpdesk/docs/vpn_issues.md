# VPN Troubleshooting Guide

## VPN Not Connecting
**Common Causes**
- Incorrect credentials
- Expired VPN certificate
- Firewall blocking UDP ports 500 or 4500

**Fix**
1. Confirm credentials.
2. Check certificate expiration.
3. Ensure firewall rules allow VPN ports.
4. Restart VPN client.

## VPN Connects but No Access
- DNS misconfiguration
- Incorrect gateway routing
- Split tunneling disabled
