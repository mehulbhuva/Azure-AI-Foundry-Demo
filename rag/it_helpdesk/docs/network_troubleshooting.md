# Network Troubleshooting Guide

## Issue: Slow Wi-Fi
**Possible Causes**
- High network traffic
- Outdated network adapter drivers
- Distance from access point

**Resolution Steps**
1. Restart the router.
2. Update the network adapter driver.
3. Move closer to the router.
4. Test on a secondary device.

## Issue: No Internet on Wired Connection
- Check Ethernet cable connection
- Verify network switch status
- Check for DHCP server availability
